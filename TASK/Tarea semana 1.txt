Subversionador


Es un software que nos permite versionar algun documento, otro software o lo que esté en él.



Administrador de versiones


Es un programa que se encarga de gestionar ágilmente los cambios en el código fuente de los programas y poder revertirlos.



Git


Sistema de control de versiones encargado del mantenimiento eficiente y confiable de versiones de aplicaciones cuando estas tienen un gran número de archivos de código fuente.



Mercurial


Es un sistema de control de versiones multiplataforma, implementado principalmente en Python. 
Diseñado principalmente para linea de comandos, gestión robusta de archivos, sin necesidad de un servidor.



Subversión


Es un proceso por el que los valores y principios de un sistema establecido se invierten.



Github


Es un programa para guardar proyectos utilizando el sistema de control de versiones Git.



Bitbucket


Es un servicio de alojamiento basado en web, para los proyectos que utilizan el sistema de control de revisiones Mercurial y Git.
